Pseudo-clases

Pseudo-clases :active

Se activa cuando el usuario pulsa el elemento ejemplo:
a:active { background-color: red;}

Pseudo-clases :focus

Se activa cuando tiene el foco sobre el elemento.
ejemplo: .input:focus { border: 2px red;}

Pseudo-elementos

Pseudo-elemento ::first-letter

Primera letra de la primera línea de texto de un elemento

Pseudo-elemento ::first-line

Primera línea de texto de un elemento

Pseudo-elemento ::before

Añade contenido al principio del documento

Pseudo-elemento ::after

Añade contenido al final del documento

Pseudo-elemento ::selection

Toma la porción del texto que se está seleccionando por el usuario y le aplica los estilos que se le asignen al pseudo-elemento.
